Solution

A queue is a first in, first out (FIFO) data structure, while a stack is a last in, first out (LIFO) data structure. We will use two stacks,Â stack1Â andÂ stack2, to preserve the FIFO property of a queue.

  • Push():Â Whenever a new element is to be pushed, we will pop all elements fromÂ stack1Â and push intoÂ stack2Â so that the latest element is pushed to the bottom ofÂ stack1. We will then pop the elements fromÂ stack2Â and push back intoÂ stack1, preserving the FIFO order.
  • Pop():Â Since the order of insertion was preserved duringÂ Push(), we will simplyÂ Pop()Â from the top ofÂ stack1.
  • Empty():Â SinceÂ stack1Â contains all the elements, its size is checked to see if the queue is empty.
  • Peek():Â We will return the top ofÂ stack1, which is the front of the queue, because the FIFO order was preserved.

class MyQueue {
    private Stack stack1;
    private Stack stack2;

    // constructor to initialize two stacks
    public MyQueue() {
        stack1 = new Stack();
        stack2 = new Stack();
    }

    public void push(int x) {
        while (!stack1.isEmpty()) {
            stack2.push(stack1.pop());
        }
        stack1.push(x);
        while (!stack2.isEmpty()) {
            stack1.push(stack2.pop());
        }
    }

    public int pop() {
        return stack1.pop();
    }

    public int peek() {
        return stack1.top();
    }

    public boolean empty() {
        return stack1.isEmpty();
    }
}
